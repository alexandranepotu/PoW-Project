1. Introducere

1.1 Scopul Documentului

Acest document descrie cerințele pentru platforma web PoW (Pet Adoption on Web), un sistem complet pentru gestionarea adopțiilor de animale de companie, cu funcționalități moderne de management, comunicare și administrare.

1.2 Domeniul de Aplicare

PoW este destinat atât proprietarilor individuali de animale, cât și organizațiilor de protecție a animalelor, oferind:

  • Gestionarea completă a animalelor de companie
  • Sistem de aplicații pentru adopție
  • Comunicare între utilizatori
  • Administrare avansată
  • Urmărirea sănătății și îngrijirii animalelor

4. Cerințe Non-Funcționale

4.1 Performanță

  • Timp de răspuns API sub 2 secunde pentru operații normale
  • Suport pentru minimum 100 utilizatori concurenți
  • Upload-ul de fișiere media până la 10MB

4.2 Securitate

  • Protecție împotriva atacurilor XSS
  • Prevenirea SQL Injection prin prepared statements
  • Criptarea parolelor cu algoritmi siguri
  • Validarea și sanitizarea tuturor input-urilor

4.3 Usabilitate

  • Interfață intuitivă și user-friendly
  • Suport pentru temă închisă/deschisă
  • Design responsive pentru toate dispozitivele
  • Navigare simplă și logică

4.4 Compatibilitate

  • Suport pentru browsere moderne (Chrome, Firefox, Safari, Edge)
  • Compatibilitate cu PHP 8.0+
  • Suport PostgreSQL 14+

7. Securitate

7.1 Autentificare și Autorizare

  • JWT (JSON Web Tokens) pentru autentificare
  • HttpOnly cookies pentru stocarea securizată a token-urilor
  • Middleware pentru verificarea autorizării
  • Role-based access control (admin, user)

7.2 Protecția Datelor

  • Criptarea parolelor cu password_hash() PHP
  • Prepared statements pentru prevenirea SQL Injection
  • Sanitizarea și validarea input-urilor
  • Escaparea output-ului pentru prevenirea XSS

7.3 Securitatea Fișierelor

  • Validarea tipurilor de fișiere la upload
  • Limitarea mărimii fișierelor (10MB max)
  • Stocarea securizată în directoare protejate
  • Generarea de nume unice pentru fișiere
